## Fonts
- Get the Fonts [here](https://www.jetbrains.com/lp/mono/) & [here](https://github.com/ryanoasis/nerd-fonts/releases/download/v3.2.1/JetBrainsMono.zip) 
- Move files to ``` ~/.local/share/fonts ```, create if doesn't exist

- Run ```fc-cache -f -v ``` 

- Done
